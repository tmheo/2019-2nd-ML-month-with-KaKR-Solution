# 제품 정의서

## 프로젝트 이름
2019년 2월 KaKR ML 대회 - 주택 가격 예측 솔루션

## 프로젝트 설명

이 프로젝트는 Kaggle의 "2019 2nd ML month with KaKR" 대회에 참여한 주택 가격 예측 머신러닝 솔루션입니다. 미국 에임스(Ames, Iowa) 주택 데이터셋을 사용하여 주택 판매 가격을 정확하게 예측하는 것을 목표합니다.

다단계 앙상블(Stacking Ensemble) 기법을 활용하여 여러 머신러닝 모델의 예측력을 조합하며, 지리적 특성을 활용한 혁신적인 특성 공학(Feature Engineering)으로 높은 예측 정확도를 달성했습니다.

**대회 성과**
- 공개 순위(Public): 98,316점 (98.3%)
- 최종 순위(Private): 99,336점 - **11위 입상**

## 대상 사용자

1. **데이터 과학자**: 앙상블 기법과 고급 특성 공학 방법론 학습

5. **소프트웨어 개발자**: 데이터 파이프라인 구축 및 모델 통합 이해

## 핵심 기능

### 1. 지리 데이터 기반 탐색적 데이터 분석(EDA)
- 주택 위치 정보(위도/경도)의 지리적 특성 분석
- 시각화를 통한 지역별 가격대 패턴 파악
- 통계 기반 이상치(Outlier) 탐지 및 처리

### 2. 고급 특성 공학(Feature Engineering)
- **시간 특성**: 건축 년도, 리모델링 년도, 판매 월/년
- **비율 특성**: 건축 면적 대비 가격, 욕실 대 침실 비율
- **지리적 특성**:
  - 주성분 분석(PCA)을 이용한 좌표 압축
  - K-Means 클러스터링으로 지역 그룹화
  - 방위각(Bearing Angle) 계산
- **그룹 통계**: 지역별 가격 통계(평균, 중앙값, 표준편차)
- **비선형 변환**: Box-Cox 변환으로 분포 정규화
- **스케일링**: RobustScaler를 이용한 정규화

### 3. 이웃 특성 생성(Neighbor Features)
- **거리 기반 이웃**: 반경 1km, 3km, 5km 내 주택 정보
- **K-최근접 이웃**: 가장 가까운 5개, 10개, 20개 이웃
- Haversine 공식으로 구면 거리(Great-circle Distance) 계산
- 이웃 주택의 통계 정보 집계

### 4. 다단계 앙상블 모델(Multi-Stage Stacking)
- **Stage 1 기본 모델(Base Models)**: 8~10개의 다양한 머신러닝 모델
  - 선형 모델: ElasticNet, Lasso, Ridge
  - 트리 기반: RandomForest, ExtraTrees
  - 부스팅: XGBoost, LightGBM, CatBoost
  - 신경망: Keras Dense 신경망, Embedding 층 활용
- **Stage 2 메타 학습**: 기본 모델의 Out-of-Fold 예측을 입력으로 사용
- **최종 앙상블**: 메타 모델을 통한 최종 예측

### 5. 모델 최적화 및 검증
- K-Fold 교차 검증(K-Fold Cross Validation)
- 프로그레시브 디스클로저(Progressive Disclosure)를 통한 모델 성능 추적
- RMSE(Root Mean Squared Error) 지표를 통한 성능 측정
- 메모리 최적화로 대용량 데이터셋 효율적 처리

## 기술적 특징

### 혁신적 측면
1. **지리적 특성의 창의적 활용**: 단순 좌표가 아닌 PCA, K-Means, 방위각 등 다양한 방식으로 처리
2. **자동 메모리 최적화**: `reduce_mem_usage()` 함수로 대용량 데이터 메모리 효율화
3. **다중 앙상블 아키텍처**: 단순 투표(Voting)가 아닌 2단계 Stacking으로 고차원적 조합
4. **특성 정규화**: 왜도(Skewness) 보정으로 데이터 분포 정규화
